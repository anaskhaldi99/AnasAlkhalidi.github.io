GitHub Recommender

GitHub Recommender is a project that recommends Github users OR GitHub repositories to another GitHub user based on his given starring activities on the website. The data about the GitHub users that I analyzed here was given to me in a text file. Based on this data, I was able to create a recommendation application that shows the recommended GitHub users or repositories for a selected user(s).

University Major Recommender for Prospective Undergraduates

This project uses web scraping (BeautifulSoup and urllib2) libraries to scrape data from a university (Istanbul Sehir University) website and extract all available UNDERGRADUATE courses. It then recommends the most suitable departments for the user, based on the keywords they type down in the "My Interests" box shown in the displayed GUI.

The program displays the most recommended undergraduate majors in decreasing order for the user.
