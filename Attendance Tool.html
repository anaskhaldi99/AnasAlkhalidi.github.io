An automated tool that allows track of students' attendance in a classroom.

Attendance Tool

This program allows the user to register attended students in a classroom by importing an Excel sheet that includes all information about students in a particular class, and then selecting the attendees. The user then chooses the section of the class, the current week of the semester, and finally exports the attended students' data in a file of his choice (.csv, .txt., or Excel file)
