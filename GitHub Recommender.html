GitHub Recommender

GitHub Recommender is a project that recommends Github users OR GitHub repositories to another GitHub user based on his given starring activities on the website. These two options of recommendation are shown from the GUI displayed above. Moreover, the user was able to choose the preferred distance metric, Pearson or Euclidean.

The user was also given the flexibility to choose the preferred programming language if needed. This filter allows him/her to search for repositories that are dedicated to a specific coding language.

The data about the GitHub users that I analyzed here was given to me in a text file. Based on this data, I was able to create a recommendation application that shows the recommended GitHub users or repositories for a selected user(s).
