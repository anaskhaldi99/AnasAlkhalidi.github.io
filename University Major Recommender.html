Undergraduate Major Recommender

A tool that recommends the most suitable university major based on given interests.

Undergraduate Major Recommender

This project uses web scraping (BeautifulSoup and urllib2) libraries to scrape data from a university (Istanbul Sehir University) website and extract all available UNDERGRADUATE courses. It then recommends the most suitable departments for the user, based on the keywords they type down in the "My Interests" box shown above in the displayed GUI.

The three ranking measures used are:

  • Text Based: The major that contains the input keyword(s) the most
  • Curriculum Based: The major that contains the maximum amount of distinct semesters that the input keywords repeat in
  • Hub-Score based: This computes the average of the total number of departments that share the same keyword(s), multiplied by the counts that input keyword(s) is repeated between the two departments.

The program then displays the most recommended undergraduate majors in decreasing order for the user.
