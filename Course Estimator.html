Department Predictor

A tool that predicts the designated department for a particular university course.

Department predictor

This project uses web scraping (BeautifulSoup and urllib2) to retrieve all undergraduate courses offered from a university (Istanbul Sehir University) and train the data to correctly predict the right department for a particular selected course.

When the user selects from the first listbox titled "Individual Courses", they will get three results as shown in the GUI above. Each prediction of the three will have a score. This score reflects the program's estimation after training. The colors reflect the correct and the incorrect department for the course selected. Green reflects the correct course in reality, and red otherwise. This means that if the program gives the highest score to the green department code, then the data has been trained well. otherwise, the data will be required to undergo more training.

Furthermore, the program allows users to perform an accuracy analysis report for each department listed. When the user selects one of the departments from the second listbox (titled "Accuracy Analysis"), the textbox on the right will show a report with the following information regarding the selected department:

  • Accuracy: this is computed as the percentage of the accurately predicted courses for that department.
  • Total Number of Courses: The total number of offered courses in Spring semester under this department.
  • Correctly Predicted: Amount of times the department was correctly predicted by different courses
  • Incorrectly predicted: Amount of times the department was incorrectly predicted by different courses. These incorrect predictions will also be displayed below as shown in the GUI
