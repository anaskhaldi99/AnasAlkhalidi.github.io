Loan Data Preprocessing for Credit Risk Analysis

This project focuses on using NumPy functions to clean and preprocess real life data of loan accounts. The tasks done for this project include:

  • Checking for incomplete data
  • filling necessary missed data
  • Splitting and stacking of datasets
  • Data type conversions
  • Performing necessary currency conversions
  • Manipulation of String and Numeric datatypes

After organizing, cleaning, and preprocessing the data, it was then exported to a csv file to be stored and used later by data scientists.
